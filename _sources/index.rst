Optifik Documentation
=====================

Welcome to the documentation for **Optifik**,
a Python library for deducing a thickness
from an interferometric spectrum.


Presentation
------------

We implemented three different methods,
each of them being adapted to a range of thicknesses.

- FFT
- Minima and maxima
- Scheludko

The physical grounds behind is explained in an article
to appear soon!
